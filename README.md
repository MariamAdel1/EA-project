# 🚦 Smart Traffic Signal Optimization using Evolutionary Algorithms

## Overview

This project focuses on improving traffic flow at a road intersection using **Evolutionary Algorithms (EA)** and traffic simulation techniques.

The system simulates vehicles moving through a multi-lane intersection and applies optimization methods to improve:

* Vehicle waiting time
* Traffic congestion
* Signal timing efficiency
* Overall traffic throughput

The project combines:

* **Python**
* **Pygame Simulation**
* **Traffic Signal Control**
* **Evolutionary Optimization Techniques**

---

# Project Goals

The main objectives of this project are:

* Simulate realistic traffic movement at an intersection
* Reduce average vehicle waiting time
* Increase the number of vehicles passing successfully
* Dynamically optimize traffic signal durations
* Study how Evolutionary Algorithms can improve traffic systems

---

# 🧠 Algorithms Used

This repository explores optimization techniques such as:

## Evolutionary Algorithms (EA)

Evolutionary Algorithms imitate natural evolution to search for better solutions.

Possible techniques used:

* Genetic Algorithm (GA)
* Particle Swarm Optimization (PSO)


These algorithms optimize:

* Green signal duration
* Traffic phase scheduling
* Vehicle flow efficiency

---

# Simulation Features

The simulation includes:

✅ Multiple traffic signals

✅ Cars, buses, bikes, and trucks

✅ Vehicle movement and lane handling

✅ Signal transitions (Green / Yellow / Red)

✅ Waiting time calculation

✅ Vehicle counting metrics

✅ Real-time traffic visualization using Pygame

✅ Performance analysis for optimization experiments
